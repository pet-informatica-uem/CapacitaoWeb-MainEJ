Sobre

Esta capacitação foi desenvolvida para apresentar a estrutura fundamental do HTML e os principais conceitos de CSS. O objetivo é demonstrar, de forma prática, como utilizar essas ferramentas para criar páginas bem estruturadas, semânticas e preparadas para evoluções futuras.

Atividades

Criar uma página para a Capacitação da EJ utilizando HTML semântico:

  • Incluir um header com navbar utilizando ul e li para criar os links de navegação
  • Construir duas seções dentro do conteúdo principal da página:
    • Seção "Sobre": com um parágrafo
    • Seção "Atividades": com uma lista não ordenada
  • Incluir um footer com alguma mensagem sua!
  • Estilizar com o CSS posteriormente.
